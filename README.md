### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 报道流程
5. 校园公告
6. 个人中心


**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息）
3. 学生管理
4. 报道流程管理
5. 学生报到管理
6. 班级分配管理
7. 宿舍分配管理
8. 缴费信息管理
9. 学生信息管理
10. 班级信息管理
11. 宿舍信息管理
12. 系统管理（轮播图管理、校园公告）
